# 5 AWS Services Used in Ecommerce Platform

1. **AWS Lambda** - Serverless compute for backend functions (Products, Cart, Orders)
2. **AWS DynamoDB** - NoSQL database (Products, Cart, Orders tables)
3. **AWS API Gateway** - REST API endpoints with JWT authentication
4. **AWS Cognito** - User authentication (signup/login with JWT tokens)
5. **AWS S3** - Static website hosting for the React frontend

*Optional: AWS CloudFront - CDN for HTTPS access (required for mobile devices)*
